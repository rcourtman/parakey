Text correction portability release. Adds import, export, sharing,
and optional file-based sync for saved text corrections. No migration
steps — `brew upgrade --cask parakey` or the in-app **Update now…**
button handles everything.

## Text Corrections

- **Import and export** — save corrections as a `.parakey-corrections`
  file, then bring them into another Mac with merge or replace.
- **Native sharing** — share the corrections file from the macOS share
  sheet, including AirDrop when available.
- **Optional sync file** — point Parakey at a corrections file in
  iCloud Drive, Dropbox, Syncthing, or another synced folder. Parakey
  reads and writes that local file; the folder provider handles
  moving it between Macs.
- **Double-click import** — `.parakey-corrections` files are registered
  as Parakey documents and can be opened with the app.

## Privacy

- Sync does not add a Parakey account, backend, or new direct network
  call. Correction files are readable JSON so they remain portable and
  auditable.
