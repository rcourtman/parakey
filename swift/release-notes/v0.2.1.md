Polish release. Menu tightening, sharper README, smaller Settings
submenu. No migration steps — `brew upgrade --cask parakey` or the
in-app **Update now…** button handles everything.

## Menu

- **Most recent transcript is back inline** — one click to copy
  what you just dictated, instead of hovering into the Recent
  submenu. The submenu still appears for older transcripts when
  there's more than one.
- **Settings is leaner** — the redundant "Check for updates now…"
  action was removed; the 6-hour automatic poll handles everything,
  and quitting + relaunching forces a fresh check via the
  30-seconds-after-launch initial poll. The remaining toggle now
  reads "Check for updates automatically" so its purpose is
  unambiguous.
- **Quit Parakey row alignment** — macOS Tahoe was decorating it
  with a destructive-action glyph that pushed the row out of line
  with the others. Routed through our own selector; flush-left now.

## README

- New blockquote stat-line in the hero — `~100 ms transcription
  · 2.2 MB download · ~80 MB RAM · 0% CPU between dictations` — so
  the speed and lightweight claims land in the first paragraph.
- "Fast" + "Lightweight" bullets carry the same numbers in prose,
  with a Safari-tab anchor for the RAM figure.
- "First launch" and "Permissions" promoted to top-level sections
  (they apply to every install path, not just contributors building
  from source).
- Accuracy fix: README no longer claims the hotkey is suppressed at
  the OS level (the event tap is listen-only, so the hotkey
  behaves normally in whatever app is focused while held).
- AI-assistant install prompt wrapped in a `<details>` so the long
  code block doesn't dominate the Install section.

## Plumbing

- `ship-swift.sh` now reads release notes from
  `swift/release-notes/v<version>.md` when present, falling back to
  the previous git-log auto-generation otherwise. (That's how
  you're reading this right now.)
- `hero.svg` and `social-preview.svg` gain matching stat rows so
  the hard numbers are visible on share-card previews too.
