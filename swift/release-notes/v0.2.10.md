## What's Changed

- **Streamlined Permission Onboarding:** Added automatic deep-linking for **Input Monitoring** and **Microphone** permissions. If a permission is missing or was previously denied, clicking "Grant" now opens the exact sub-pane in macOS System Settings so you don't have to hunt for it.
- **Actionable Setup Guidance:** Rewrote all checklist descriptions to provide clear instructions on what switch to toggle or button to click.
- **Self-Healing Assistance:** Added a visible tip explaining that clicking "Try Again" triggers Parakey's built-in TCC database reset to resolve stuck macOS permission states.
- **Layout and Alignment Fixes:** Resolved an AppKit auto-layout bug where long description texts caused text elements to shift and crop at the screen edges.
