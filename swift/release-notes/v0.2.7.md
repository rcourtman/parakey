## What's Changed

- Added a manual **Check for Updates...** action to the menu.
- Added visible results for update checks, including up-to-date,
  failed-check, and manual-update-needed states.
- Hardened the in-app Homebrew updater so it refreshes the Parakey tap,
  fetches the tap-qualified cask, and uses tap-qualified upgrade and
  reinstall commands.

## Updating From v0.2.6

If the in-app **Update Now...** action does not relaunch Parakey or
does not install the latest version, run:

```sh
brew update && brew upgrade --cask rcourtman/parakey/parakey
```
