Text correction release. Adds a local corrections dictionary for
recurring words or phrases the speech model consistently mishears.
No migration steps — `brew upgrade --cask parakey` or the in-app
**Update now…** button handles everything.

## Text Corrections

- **Local correction dictionary** — add replacement pairs from
  **Settings → Text Corrections**. Put the incorrect text Parakey
  typed under **Typed**, and the desired output under **Paste**.
- **Whole-word and phrase matching** — corrections apply after
  transcription and before paste/history, with longest matches
  winning when entries overlap.
- **Private by default** — corrections are stored in local
  preferences. Audio still stays on-device, transcripts are still
  never written to disk, and logs only record how many corrections
  were applied.

## Polish

- The correction editor uses a fixed AppKit layout so the text fields
  render reliably inside the menu-triggered dialog.
- Dialog copy now uses `Typed` → `Paste` wording and a clearer
  `clawed` → `Claude` example.
