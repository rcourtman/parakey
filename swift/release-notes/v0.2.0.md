**Parakey is now a native Swift app.** Same UX — push-to-talk, menu
bar, no preferences window — rebuilt on the Apple Neural Engine via
[FluidAudio](https://github.com/FluidInference/FluidAudio) +
[Parakeet TDT v3](https://huggingface.co/nvidia/parakeet-tdt-0.6b-v3).

- **~100 ms** from key release to pasted text (down from ~180 ms)
- **2.2 MB** download (down from ~150 MB)
- Two hardened-runtime entitlements instead of six
- No embedded interpreter, no JIT, no library-validation override

## Upgrading from v0.1.x

`brew upgrade --cask parakey` or the in-app **Update now…** button
handles the swap. Settings, granted permissions, and Skip-version
state carry over cleanly — same bundle id, same Developer ID cert.

**Heads up: v0.2.0 requires macOS 26 (Tahoe).** If you're on macOS
13, 14, or 15, the upgrade will refuse with a brew error — stay on
v0.1.6 until you can upgrade macOS.

### Reclaiming disk space (optional)

The old MLX model cache from v0.1.x is no longer used. ~600 MB:

```sh
rm -rf ~/.cache/huggingface/hub/models--mlx-community--parakeet-tdt-0.6b-v2
```

If you ever installed v0.1.x from a source clone via `install.sh`,
the LaunchAgent it created is also unused now:

```sh
launchctl bootout gui/$(id -u) ~/Library/LaunchAgents/com.local.parakey.plist 2>/dev/null
rm -f ~/Library/LaunchAgents/com.local.parakey.plist
```
